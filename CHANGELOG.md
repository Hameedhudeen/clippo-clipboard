# Changelog

All notable changes to Clippo will be documented in this file.

## Unreleased

- No changes yet.

## 0.1.2-alpha - 2026-06-02

- Added first-run startup opt-in prompts for macOS, Windows, and Linux.
- Improved resident background lifecycle behavior across menu bar, tray, and Linux background monitor paths.
- Added alpha packaging support for Linux downloads, macOS zipped app bundles, and Windows self-contained zip builds.
- Updated release documentation and version tracking for alpha releases.

## 0.1.1-pre-alpha - 2026-06-02

- Published Linux pre-alpha artifacts for tester feedback.
- Added Linux AppImage, Debian package, Flatpak local repo archive, and checksum release assets.

## 0.1.0-pre-alpha - 2026-06-02

- Started planning and repository scaffolding.
- Added the initial Clippo workflow, platform, and validation baseline.
- Added a shared-core benchmark snapshot from 2026-06-01: add 1,000 history items in 2,153 us, search 1,000 items in 174 us, save 1,000 items to JSON in 331 us, and load 1,000 items from JSON in 205 us.
