# FAQ

## Is Clippo ready to use?

Not yet. Clippo is pre-alpha.

## Will Clippo support Linux Wayland?

Wayland is a v1 target, but support depends on desktop portals and compositor behavior.

## Does Clippo collect telemetry?

No telemetry is planned by default.

## Why does paste automation need permissions?

Automatic paste interacts with the active application after a user chooses a clipboard item. Each OS controls that differently.
