# Roadmap

The full implementation backlog lives in `tasks.md`. This page is a shorter public roadmap for GitHub readers.

GitHub-ready issue drafts live in `docs/ROADMAP_ISSUES.md`. Milestone and project-board planning lives in `docs/MILESTONES.md`.

## v0.1

- Repository scaffolding.
- Rust shared core workspace.
- Initial clipboard history model.

## v0.2

- Tested history, search, settings, and persistence logic.
- Platform service traits for native shells.

## v0.3

- One native shell usable end to end.

## v0.4

- macOS, Windows, and Linux shells can open, search, copy, and paste history items.

## v0.5

- Core Maccy workflow parity using OS-appropriate equivalents.

## v1.0

- Packaged releases, documented limitations, privacy review, and external beta feedback.
